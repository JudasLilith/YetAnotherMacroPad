# YetAnotherMacroPad
A macropad to be used for Godot and Freecad. It has a save key, a delete key, and two custom keybinds.

# BOM
You'll need:
* 4 x Cherry MX keys
* 4 x Any Keycaps you like!
* 4 x M3x12mm Bolts
* 3 x M2x6mm Bolts
* 3 x M2 nuts
* 2 x SK6812Mini-LEDs
* 1 x custom pcb board
* 1 x Casing
* 1 x Casing Cover
